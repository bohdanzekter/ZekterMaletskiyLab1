Написати клас функціональних об’єктів. Об’єкт класу роздає кістки доміно (стандартна
кістка доміно містить два натуральних числа від 0 до 6 включно, можливо однакові; наші кістки доміно
містять два натуральних числа від 0 до включно, можливо, однакові). Оператор () повертає чергову
випадкову кістку з поточного залишку.
Кожна «роздача» починається з повного комплекту кісток і продовжується поки чергову кістку
можна прикласти до витягнутих раніше (див. правила гри в доміно; +якщо кістку можна прикласти
обома кінцями, то вона прикладається найменшим числом). Розміром «роздачі» будемо вважати
кількість розданих кісток на якій вона зупинилася.
